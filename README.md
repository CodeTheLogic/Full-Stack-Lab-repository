# Full Stack Development Lab  
## Assignment 03 – Student Registration Form with Validation  

### 📌 Objective
To design a student registration form and implement client-side validation using JavaScript, DOM manipulation, and jQuery.

---

### 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- jQuery

---

### ✨ Functionalities Implemented

#### ✔ Form Validations
- All fields mandatory validation  
- Email validation using Regular Expressions  
- Phone number validation (10 digits only)  
- Password strength validation  
- Confirm password matching  

---

#### ✔ DOM Manipulation
- Access elements using:
  - getElementById
  - getElementsByClassName
  - getElementsByTagName
- Change text using innerHTML
- Change CSS properties dynamically
- Change image source on button click
- Add and delete text nodes dynamically

---

#### ✔ jQuery Features
- Change button text dynamically
- Modify background image using jQuery CSS property
- Access form data using jQuery
- Add attributes dynamically

---

### 🎨 UI Features
- Styled registration form
- Responsive layout
- Interactive buttons
- Dynamic visual feedback

---

### ✅ Learning Outcome
- Learned JavaScript validation techniques  
- Understood DOM manipulation  
- Implemented jQuery operations  
- Developed interactive web forms
