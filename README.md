# 💻 Assignment 04 – PHP & MySQL CRUD Application

## 📌 Aim
To design and develop a dynamic web application using PHP and MySQL that performs CRUD operations.

---

## 🎯 Objectives
- Understand server-side scripting using PHP
- Learn how to connect PHP with MySQL database
- Perform CRUD operations:
  - Insert
  - Update
  - Delete
  - Search

---

## 🧠 Technologies Used
- PHP
- MySQL
- HTML
- CSS
- XAMPP Server

---

## ⚙️ Features
- Add new records to database
- Update existing records
- Delete records
- Search records
- Display data in tabular format
